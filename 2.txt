Repository : مستودع الاكواد كاملة
كل بروجكت اله ريبو خاص فيه

Branch : فرع من الريبو ممكن يكون جزء من الكود بتعدل عليه ,كل برانش له وضيفة معينة يعني اما اضافة شيئ للكود...

local Repo : الريبو الخاص فيك الي هو حهازك

Remote Repo : الموقع الخاص بالشركة ممكن يكون سيرفر خاص فيهم او مثلا سيرفر الجتهوب

commit : تعتبر شيك بوينت للكود تبعك 

clone : استنساخ الكود اما على اللوكل ريبو او على الريموت 

pull : سحب كود

push : رفع كود 

pull request : اخبار الاخرين انك رح ترفع الكود عللى الريبو مشان اذا فيه خطا او اشي
